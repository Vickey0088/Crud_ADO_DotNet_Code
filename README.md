📊 ASP.NET MVC CRUD Application using ADO.NET
📌 Project Overview

This project is a CRUD (Create, Read, Update, Delete) web application built using ASP.NET MVC and ADO.NET. The application demonstrates how to perform direct database operations with Microsoft SQL Server using ADO.NET components such as SqlConnection, SqlCommand, SqlDataReader, and SqlDataAdapter.

The goal of this project is to showcase database connectivity, data manipulation, and MVC architecture while following clean coding practices. 🚀

🛠️ Technologies Used
💻 ASP.NET MVC
⚙️ C#
🗄️ Microsoft SQL Server (MSSQL)
🔗 ADO.NET
🔍 SQL Queries
🎨 Bootstrap
📄 Razor Views
✨ Features
🔄 CRUD Operations

The application allows users to perform full CRUD operations:

➕ Create new records
📖 Read or view all records
✏️ Update existing records
❌ Delete records from the database

All operations interact with SQL Server using ADO.NET.

🗄️ ADO.NET Implementation

The project uses core ADO.NET components for database operations:

🔹 SqlConnection – Establish connection with SQL Server
🔹 SqlCommand – Execute SQL queries and stored procedures
🔹 SqlDataReader – Read data from the database
🔹 SqlDataAdapter – Fill datasets and manage data

This approach provides fine control over database interactions without using ORM frameworks.

🏗️ Project Architecture

The application follows the ASP.NET MVC architecture:

Models
   ↓
Controllers
   ↓
Views
   ↓
SQL Server Database
📦 Models represent database entities
🎯 Controllers handle user requests and business logic
🎨 Views display data using Razor syntax
🗄️ SQL Server stores application data
